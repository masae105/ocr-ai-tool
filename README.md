# OCR AI Tool

## 📌 概要

PDFや画像から文字を自動認識（OCR）し、抽出したデータをExcelへ自動入力するツールです。

請求書・領収書などの紙データを手入力する作業を削減し、業務効率化を目的として開発しています。

---

## 🎯 開発目的

日常業務では、PDFや画像化された書類から必要な情報を手入力する作業が多く発生します。

そこで、

* PDF・画像の読み取り
* OCRによる文字抽出
* データ整理
* Excelへの自動入力

までをPythonで自動化するツールの開発に取り組みました。
